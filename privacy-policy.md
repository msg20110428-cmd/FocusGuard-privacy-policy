# FocusGuard 개인정보처리방침

시행일: 2026년 5월 4일

FocusGuard는 사용자의 숏폼 콘텐츠 사용 시간을 관리하고, 경고/개입, 리포트, 계정 동기화 기능을 제공하기 위해 필요한 범위에서 개인정보를 처리합니다. 본 개인정보처리방침은 FocusGuard Android 앱과 FocusGuard 서버에 적용됩니다.

## 1. 처리하는 개인정보 항목

FocusGuard는 서비스 제공을 위해 다음 정보를 처리할 수 있습니다.

### 계정 및 로그인 정보

- 이메일 주소
- 비밀번호의 단방향 해시값
- Google 또는 Kakao 소셜 로그인 제공자 정보
- 소셜 로그인 제공자가 전달하는 계정 식별자, 이메일, 프로필 정보
- 앱에서 생성한 기기 식별자, 기기 모델명
- 로그인 토큰, 세션 정보

FocusGuard는 사용자의 비밀번호 원문을 저장하지 않습니다.

### 이메일 인증 정보

- 이메일 주소
- 인증 코드 발송 및 검증 상태
- 인증 코드 생성/만료 시각

인증 코드는 회원가입 및 계정 확인 목적으로만 사용됩니다.

### 앱 사용 관리 정보

- 숏폼 앱 또는 숏폼 피드 감지 기록
- 앱 패키지명, 사용 시간, 세션 시간, 감지 시각
- 경고 단계, 차단/오버레이/흑백 전환 상태
- 사용자가 설정한 제한 시간, 튜토리얼 완료 여부, 앱 설정값
- 자기 점검 또는 리포트에 사용자가 입력한 내용

### 접근성 및 사용정보 접근으로 처리되는 정보

FocusGuard는 숏폼 피드 감지와 시간 기반 개입을 위해 Android 접근성 서비스 및 사용정보 접근 권한을 사용할 수 있습니다. 접근성 서비스는 현재 화면의 앱 전환, 화면 상태, 스크롤 이벤트, 일부 화면 구성 정보를 처리할 수 있으며, 필요한 경우 숏폼 피드 여부를 판단하기 위해 화면 정보를 일시적으로 확인할 수 있습니다.

이 정보는 숏폼 사용 감지, 경고 표시, 오버레이 해제, 흑백 전환 복구 등 앱의 핵심 기능을 제공하기 위한 목적으로만 사용됩니다. FocusGuard는 접근성 권한을 키 입력 감시, 금융 정보 수집, 메시지 내용 수집, 연락처 수집, 광고 추적 목적으로 사용하지 않습니다.

### 리포트, 랭킹, 친구 기능 정보

- 일간/주간/월간 사용 통계
- 리포트 생성 및 열람/미열람 상태
- 랭킹 점수, 친구 관계, 친구 요청 상태
- 닉네임 또는 프로필 표시 정보

### 고객 지원 및 문의 정보

사용자가 이메일로 문의하는 경우 이메일 주소, 문의 내용, 첨부 정보가 처리될 수 있습니다.

## 2. 개인정보 처리 목적

FocusGuard는 수집한 정보를 다음 목적에 사용합니다.

- 회원가입, 로그인, 이메일 인증
- Google/Kakao 소셜 로그인 연동
- 숏폼 콘텐츠 사용 시간 측정
- 3단계 경고, 오버레이, 흑백 전환 등 사용 제한 기능 제공
- 일간/주간/월간 리포트 및 리포트 보관함 제공
- 랭킹, 친구, 알림 기능 제공
- 부정 이용 방지, 계정 보호, 오류 진단
- 고객 문의 응대 및 서비스 개선

## 3. 개인정보의 제3자 제공 및 처리 위탁

FocusGuard는 법령에 따른 경우를 제외하고 사용자의 개인정보를 임의로 판매하거나 제3자에게 제공하지 않습니다. 다만 서비스 운영을 위해 다음 외부 서비스를 사용할 수 있습니다.

- Google Sign-In: Google 계정 로그인 및 인증
- Kakao Login: Kakao 계정 로그인 및 인증
- SendGrid: 이메일 인증 코드 및 안내 메일 발송
- Railway: 서버 호스팅
- MySQL 데이터베이스: 계정 및 서비스 데이터 저장
- Google Gemini API: 리포트 생성 또는 코칭 기능이 활성화된 경우 사용 통계와 입력 내용을 기반으로 분석 결과 생성

외부 서비스는 각 제공자의 개인정보처리방침 및 보안 정책에 따라 정보를 처리할 수 있습니다.

## 4. 개인정보 보관 기간

FocusGuard는 개인정보를 서비스 제공에 필요한 기간 동안 보관합니다.

- 계정 정보: 회원 탈퇴 또는 삭제 요청 시까지
- 이메일 인증 코드: 인증 목적 달성 또는 만료 후 삭제
- 사용 기록 및 리포트: 계정 유지 기간 동안 보관하며, 삭제 요청 시 삭제
- 서버 로그: 보안 및 장애 대응을 위해 필요한 기간 동안 보관
- 법령상 보관 의무가 있는 정보: 해당 법령에서 정한 기간 동안 보관

사용자가 앱을 삭제하면 기기 내 로컬 데이터는 삭제될 수 있으나, 서버에 저장된 계정 및 리포트 데이터는 별도 삭제 요청 전까지 남아 있을 수 있습니다.

## 5. 개인정보의 파기

보관 목적이 달성되거나 보관 기간이 종료된 개인정보는 복구하기 어려운 방식으로 삭제합니다. 전자 파일은 안전한 방법으로 삭제하고, 출력물이 있는 경우 분쇄 또는 이에 준하는 방식으로 파기합니다.

- 계정 삭제 요청
- 개인정보 처리 정지 요청

## 7. 권한 사용 안내

FocusGuard는 앱 기능 제공을 위해 다음 Android 권한을 사용할 수 있습니다.

- 인터넷 접근: 서버 통신, 로그인, 이메일 인증
- 알림 권한: 경고, 리포트, 친구 요청 등 알림 표시
- 사용정보 접근: 앱 사용 시간 측정
- 접근성 서비스: 숏폼 피드 감지 및 화면 기반 개입
- 시스템 설정 변경 권한: 흑백 전환 등 집중 보조 기능

사용자는 Android 설정에서 각 권한을 언제든지 변경하거나 해제할 수 있습니다. 일부 권한을 해제하면 숏폼 감지, 경고, 흑백 전환, 리포트 생성 등 핵심 기능이 제한될 수 있습니다.

## 8. 아동의 개인정보

FocusGuard는 만 14세 미만 아동의 개인정보를 고의로 수집하지 않습니다. 만 14세 미만 사용자가 서비스를 이용하려는 경우 법정대리인의 동의가 필요합니다. 법정대리인이 아동의 개인정보 삭제를 요청하면 확인 후 필요한 조치를 진행합니다.

## 9. 개인정보 보호 조치

FocusGuard는 개인정보 보호를 위해 다음 조치를 적용합니다.

- HTTPS 기반 서버 통신
- 비밀번호 원문 미저장 및 해시 처리
- 인증 토큰 및 민감 정보의 안전한 저장
- 서버 접근 권한 제한
- 불필요한 개인정보 수집 최소화

## 10. 개인정보처리방침 변경

본 개인정보처리방침은 서비스 변경, 법령 변경, 운영 정책 변경에 따라 수정될 수 있습니다. 중요한 변경이 있는 경우 앱 또는 공지 수단을 통해 안내합니다.
